## 📄 User Stories

| ID    | Descrição                                                                                                |
| :---- | :------------------------------------------------------------------------------------------------------- |
| US-01 | Como um usuário, eu quero alternar facilmente entre as telas, para que eu possa gerenciar meu tempo e minhas obrigações sem dificuldades.                            
| US-02 | Como um usuário ativo à noite, eu quero escolher o tema do aplicativo, para que eu possa adaptar as minhas preferências e reduzir o cansaço visual.
| US-03 | Como um usuário aterefado, eu quero adicionar tarefas, para que eu possa salvar para não esquece-las.                                
| US-04 | Como um usuário organizado, eu quero excluir tarefas, para que eu possa deixar a visualização mais limpa e evitar confusões entre tarefas.                                                 
| US-05 | Como um usuário do método pomodoro, eu quero visualizar o cronômetro, para que eu saiba o tempo que ainda possuo enquanto realizo uma tarefa.                      
| US-06 | Como um novo usuário, eu quero fazer cadastro, para que eu possa salvar minhas informações e tarefas de forma segura vinculando a uma conta.                                            
| US-07 | Como um usuário recorrente, eu quero fazer login, para que eu tenha acesso as minhas tarefas e outras informações vínculadas no meu perfil.                      
| US-08 | Como um usuário registrado, eu quero atualizar minhas informações, para que eu possa sempre mantê-la atualizada.                       
| US-09 | Como um usuário focado, eu quero uma tela principal, para que eu possa ter uma visão ampla das tarefas planejadas para o meu dia.
